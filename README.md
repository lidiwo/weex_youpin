# 运行

## 安装weex开发环境（参考官网）

## 下载代码，进入根目录，安装依赖包

```
$ npm install
```

## 运行web

web工程打包

```
$ npm run build
```

构建web工程 & 启动服务

```
$ npm run dev & npm run serve
```
